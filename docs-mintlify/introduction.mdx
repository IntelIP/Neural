## Quick Example

<CodeGroup>

```python Python
from neural_sdk import NeuralSDK

# Initialize SDK
sdk = NeuralSDK.from_env()

# Stream real-time NFL markets
websocket = sdk.create_websocket()

@websocket.on_market_data
async def handle_market(data):
    if data.yes_price < 0.30:
        print(f"ðŸŽ¯ Buy opportunity: {data.ticker} @ ${data.yes_price}")

await websocket.connect()
await websocket.subscribe_markets(['KXNFL*'])
await websocket.run_forever()
```

```javascript JavaScript
import { NeuralSDK } from 'neural-sdk';

// Initialize SDK
const sdk = NeuralSDK.fromEnv();

// Stream real-time NFL markets
const websocket = sdk.createWebsocket();

websocket.onMarketData((data) => {
  if (data.yesPrice < 0.30) {
    console.log(`ðŸŽ¯ Buy opportunity: ${data.ticker} @ $${data.yesPrice}`);
  }
});

await websocket.connect();
await websocket.subscribeMarkets(['KXNFL*']);
await websocket.runForever();
```
