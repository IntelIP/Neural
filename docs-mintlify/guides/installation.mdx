## Installation Methods

<Tabs>
  <Tab title="pip (Recommended)">
    ```bash
    # Install from main branch
    pip install git+https://github.com/IntelIP/Neural-Trading-Platform.git@main
    
    # Install specific version
    pip install git+https://github.com/IntelIP/Neural-Trading-Platform.git@v1.1.0
    ```
  </Tab>
  
  <Tab title="uv (Faster)">
    ```bash
    # Install with uv for faster package management
    uv add git+https://github.com/IntelIP/Neural-Trading-Platform.git@main
    ```
  </Tab>
  
  <Tab title="Development Install">
    ```bash
    # Clone repository
    git clone https://github.com/IntelIP/Neural-Trading-Platform.git
    cd Neural-Trading-Platform
    
    # Install in editable mode
    pip install -e .
    
    # Install with development dependencies
    pip install -e .[dev]
    ```
  </Tab>
</Tabs>

## Quick Installation Script

Use our automated installation script:

```bash
# Download and run installation script
curl -sSL https://raw.githubusercontent.com/IntelIP/Neural-Trading-Platform/main/scripts/install_neural_sdk.sh -o /tmp/install_neural.sh && bash /tmp/install_neural.sh
```

## Common Issues

<AccordionGroup>
  <Accordion title="ModuleNotFoundError">
    If you see `ModuleNotFoundError: No module named 'data_pipeline'`, reinstall the latest version:
    ```bash
    pip uninstall neural-sdk -y
    pip install --force-reinstall git+https://github.com/IntelIP/Neural-Trading-Platform.git@main
    ```
  </Accordion>
  
  <Accordion title="GitHub Authentication">
    If you get authentication errors, configure Git credentials:
    ```bash
    # Using HTTPS with credentials
    git config --global credential.helper store
    
    # Or use SSH keys (recommended)
    ssh-keygen -t ed25519 -C "your-email@example.com"
    # Add public key to GitHub account
    ```
  </Accordion>
  
  <Accordion title="Python Version Issues">
    Ensure you're using Python 3.10+:
    ```bash
    python --version  # Should be 3.10 or higher
    
    # If using wrong version, create virtual environment
    python3.11 -m venv neural_env
    source neural_env/bin/activate  # On Windows: neural_env\Scripts\activate
    ```
  </Accordion>
</AccordionGroup>
