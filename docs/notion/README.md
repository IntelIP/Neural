# Notion Staging

Use this directory when Codex needs to prepare content for Notion but does not have direct Notion access in the current session.

Expected files:

- `prds/*.md`
- `sprint-briefs/*.md`
- `weekly-updates/*.md`
- `release-notes/*.md`
- `postmortems/*.md`
- `decisions/*.md`

Each file should be written as Notion-ready Markdown with short headings and no repo-local assumptions.
