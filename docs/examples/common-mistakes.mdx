## Missing Credentials

- **Symptom**: Tests skip or raise `FileNotFoundError` for secrets files
- **Fix**:
  - Set `KALSHI_API_KEY_ID` and `KALSHI_PRIVATE_KEY_BASE64` in a `secrets/.env` and export
  - Or specify `KALSHI_PRIVATE_KEY_PATH` to a PEM file

## Using Outdated APIs

- Use `TradingClient` and its stable methods directly: `place_order`, `list_markets`, `get_positions`
- Use the compatibility proxies only when you need legacy integration points: `portfolio`, `markets`, `exchange`

## Running Integration Tests Without Secrets

- **Expected**: integration tests are skipped
- **To run them**:
  - `export $(grep -v '^#' secrets/.env | xargs)`
  - `pytest -q`

## Pip vs uv

- `uv` is fast and works great for project installs:
  - `uv add neural-sdk`
  - `uv add "neural-sdk[trading]"`
  - `uv add "neural-sdk[sentiment]"`
  - `uv add "neural-sdk[deployment]"`
- `pip` works as well:
  - `pip install neural-sdk`
  - `pip install "neural-sdk[trading]"`
  - `pip install "neural-sdk[sentiment]"`
  - `pip install "neural-sdk[deployment]"`
