## üî¥ Mistake #1: Wrong Sports Tickers

### ‚ùå What People Try (That Doesn't Work)

```python
# ALL OF THESE FAIL:
source = KalshiMarketsSource(series_ticker="NFL")
source = KalshiMarketsSource(series_ticker="Pro Football")
source = KalshiMarketsSource(series_ticker="Football")
source = KalshiMarketsSource(series_ticker="FOOT")
```

**Result:** `No markets found matching criteria`

### ‚úÖ What Actually Works

```python
# CORRECT - Use the actual event ticker:
source = KalshiMarketsSource(series_ticker="KXNFLGAME")
```

### üìù Complete Reference

| Sport | ‚ùå WRONG | ‚úÖ CORRECT |
|-------|----------|-----------|
| NFL | "NFL", "Pro Football" | `KXNFLGAME` |
| NBA | "NBA", "Basketball" | `KXNBA` |
| College Football | "NCAA", "CFB" | `KXNCAAFGAME` |

<Note>
**Why This Happens:** The Kalshi website shows "Pro Football" but the API uses `KXNFLGAME`. They're different systems!
</Note>

## üü° Mistake #2: Filtering by Wrong Status

### ‚ùå The Silent Killer

```python
# This filters out 90% of games!
source = KalshiMarketsSource(
    series_ticker="KXNFLGAME",
    status="open"  # ‚Üê THE PROBLEM
)
```

**Why it fails:** Sports markets are usually `"active"`, not `"open"`!

### ‚úÖ The Fix

```python
# Get ALL games regardless of status:
source = KalshiMarketsSource(
    series_ticker="KXNFLGAME",
    status=None  # ‚Üê No filtering!
)
```

### üìä Status Types Explained

| Status | What It Means | Example |
|--------|--------------|---------|
| `"active"` | Currently trading | Live NFL game |
| `"closed"` | Trading paused | Halftime |
| `"finalized"` | Game over | Settled bets |
| `"initialized"` | Not yet trading | Future game |
| ~~`"open"`~~ | **RARELY USED** | Not for sports! |

## üîµ Mistake #3: Wrong API Environment Variables

### ‚ùå Common Typos

```bash
# WRONG variable names:
KALSHI_API_KEY=xxx           # ‚Üê Missing _ID
KALSHI_PRIVATE_KEY=xxx       # ‚Üê Should be _PATH
KALSHI_KEY_ID=xxx           # ‚Üê Missing API_
```

### ‚úÖ Correct Names

```bash
# EXACTLY these names:
KALSHI_API_KEY_ID=AKI_your_key_here
KALSHI_PRIVATE_KEY_PATH=secrets/kalshi_private.pem
```

### üîç Debug Helper

```python
import os

def check_env():
    """Check if environment is set up correctly"""
    correct = {
        'KALSHI_API_KEY_ID': os.getenv('KALSHI_API_KEY_ID'),
        'KALSHI_PRIVATE_KEY_PATH': os.getenv('KALSHI_PRIVATE_KEY_PATH')
    }

    wrong = {
        'KALSHI_API_KEY': os.getenv('KALSHI_API_KEY'),
        'KALSHI_PRIVATE_KEY': os.getenv('KALSHI_PRIVATE_KEY'),
        'KALSHI_KEY': os.getenv('KALSHI_KEY')
    }

    print("‚úÖ CORRECT Variables:")
    for key, val in correct.items():
        print(f"  {key}: {'SET' if val else 'NOT SET'}")

    print("\n‚ùå WRONG Variables (shouldn't be set):")
    for key, val in wrong.items():
        if val:
            print(f"  {key}: SET (should be {key}_ID or {key}_PATH)")

check_env()
```

## üü£ Mistake #4: Not Using Async Properly

### ‚ùå Forgetting Async/Await

```python
# THIS DOESN'T WORK:
from neural.data_collection import get_game_markets

# Wrong - not using async
df = get_game_markets("KXNFLGAME-25SEP22DETBAL")  # ERROR!
```

**Error:** `RuntimeWarning: coroutine 'get_game_markets' was never awaited`

### ‚úÖ Correct Async Usage

```python
import asyncio
from neural.data_collection import get_game_markets

# Method 1: In async function
async def get_data():
    df = await get_game_markets("KXNFLGAME-25SEP22DETBAL")
    return df

df = asyncio.run(get_data())

# Method 2: Direct run
df = asyncio.run(get_game_markets("KXNFLGAME-25SEP22DETBAL"))
```

## üü† Mistake #5: Wrong Date Format in Tickers

### ‚ùå What People Guess

```python
# All wrong date formats:
"KXNFLGAME-09-22-25-DETBAL"    # Wrong: MM-DD-YY
"KXNFLGAME-2025-09-22-DETBAL"  # Wrong: YYYY-MM-DD
"KXNFLGAME-22SEP2025DETBAL"    # Wrong: DDMMMYYYY
"KXNFLGAME-SEP22DETBAL"        # Wrong: Missing year
```

### ‚úÖ Correct Format: YYMMMDD

```python
# CORRECT: YYMMMDD format
"KXNFLGAME-25SEP22DETBAL"  # 2025, September 22
"KXNFLGAME-25OCT15LALNYK"  # 2025, October 15
"KXNFLGAME-26JAN01BUFKC"   # 2026, January 1
```

**Remember:** Year (2 digits) + Month (3 letters) + Day (2 digits)

  </Step>

### "Coroutine was never awaited"

