# üèà Sports Markets Guide

## üö® The #1 Thing You Must Know

<Note>
Sports markets use **EVENT TICKERS** not series names:
- ‚ùå WRONG: `"NFL"`, `"Pro Football"`, `"Football"`
- ‚úÖ RIGHT: `"KXNFLGAME"`

This single mistake causes 90% of "no markets found" errors.
</Note>

## üîç How to Find ANY Sports Game

### Method 1: Get All Games for a Sport

```python
from neural.data_collection import KalshiMarketsSource
import asyncio

async def get_all_nfl_games():
    """Get ALL NFL games (active, upcoming, completed)"""

    source = KalshiMarketsSource(
        series_ticker="KXNFLGAME",  # ‚Üê CORRECT ticker
        status=None,                 # ‚Üê NO filtering (critical!)
        use_authenticated=True        # ‚Üê Better data access
    )

    async with source:
        async for df in source.collect():
            print(f"Found {len(df)} NFL markets")

            # Group by status to see what's available
            print("\nMarkets by status:")
            print(df['status'].value_counts())

            # Show some games
            print("\nSample games:")
            for _, row in df.head(5).iterrows():
                print(f"  {row['ticker']}: {row['title']}")

            return df

# Run it
df = asyncio.run(get_all_nfl_games())
```

<Warning>
**COMMON MISTAKE:** Using `status="open"` filters out most games!
- Games can be: `"active"`, `"closed"`, `"finalized"`, `"initialized"`
- Use `status=None` to see everything
</Warning>

### Method 2: Find a Specific Game

```python
from neural.data_collection import get_game_markets

async def get_seahawks_cardinals():
    """Get Thursday Night Football: Seahawks at Cardinals"""

    # The event ticker format: PREFIX-DATETEAMS
    event_ticker = "KXNFLGAME-25SEP25SEAARI"
    #               ‚Üë         ‚Üë      ‚Üë   ‚Üë

            print(f"  Yes Price: ${market['yes_ask']}")
            print(f"  No Price: ${market['no_ask']}")
```

### Method 3: Search by Team Name

```python
async def find_cowboys_games():
    """Find all Dallas Cowboys games"""

    source = KalshiMarketsSource(
        series_ticker="KXNFLGAME",
        status=None
    )

    async with source:
        async for df in source.collect():

                    na=False

            print(f"Found {len(cowboys)} Cowboys markets:")
```

## üìÖ Understanding Event Ticker Patterns

Every sports event ticker follows this pattern:

```
[PREFIX]-[DATE][TEAM1][TEAM2]
```

### Decoding Examples

| Full Ticker | Breakdown | Meaning |
|------------|-----------|---------|
| `KXNFLGAME-25SEP22DETBAL` | KX-NFL-GAME + 25SEP22 + DET + BAL | NFL game on Sept 22, 2025: Detroit at Baltimore |
| `KXNFLGAME-25SEP25SEAARI` | KX-NFL-GAME + 25SEP25 + SEA + ARI | NFL game on Sept 25, 2025: Seattle at Arizona |
| `KXNBA-25OCT15LALNYK` | KX-NBA + 25OCT15 + LAL + NYK | NBA game on Oct 15, 2025: Lakers at Knicks |

### Date Format: `YYMMMDD`
- `25SEP22` = September 22, 2025
- `25OCT15` = October 15, 2025
- `25DEC25` = December 25, 2025

### Team Codes (Common NFL)
| Team | Code | Team | Code |
|------|------|------|------|
| Baltimore Ravens | BAL | Detroit Lions | DET |
| Seattle Seahawks | SEA | Arizona Cardinals | ARI |
| Dallas Cowboys | DAL | Philadelphia Eagles | PHI |
| Kansas City Chiefs | KC | Buffalo Bills | BUF |

```python
from neural.data_collection import get_live_sports
import asyncio
from datetime import datetime

async def monitor_live_nfl():
    """Get all currently active NFL games"""

    while True:
        print(f"\n[{datetime.now()}] Checking live games...")

        # Get only active NFL games
        df = await get_live_sports(sports=["NFL"])

        if not df.empty:
            print(f"Found {len(df)} live markets:")

            # Group by game
            games = df.groupby('event_ticker')
            for event, markets in games:
                print(f"\n{event}:")
                for _, m in markets.iterrows():
                    team = m['ticker'].split('-')[-1]
                    print(f"  {team}: Yes ${m['yes_ask']} / No ${m['no_ask']}")
        else:
            print("No live NFL games right now")

        # Check every 30 seconds
        await asyncio.sleep(30)

# Run for 5 minutes
asyncio.run(asyncio.wait_for(monitor_live_nfl(), timeout=300))
```

## ‚ö†Ô∏è Common Problems & Solutions

<AccordionGroup>
  <Accordion title="üî¥ No markets found">
    **Check these in order:**
    1. Are you using `KXNFLGAME` not `"NFL"`?
    2. Is `status=None` (not `"open"`)?
    3. Are you authenticated? (`use_authenticated=True`)
    4. Is the game ticker format correct? (`PREFIX-DATETEAMS`)
    5. Has the game already finished? Check without date filter
  </Accordion>

  <Accordion title="üü° Markets show wrong status">
    **Game Status Guide:**
    - `"initialized"` ‚Üí Game created, not yet trading
    - `"active"` ‚Üí Currently trading (NOT "open"!)
    - `"closed"` ‚Üí Trading paused/stopped
    - `"finalized"` ‚Üí Game over, bets settled

    **Fix:** Use `status=None` to see all statuses
  </Accordion>

  <Accordion title="üü¢ Can't find specific team">
    **Team names vary in markets:**
    - Try full name: "Baltimore Ravens"
    - Try city: "Baltimore"
    - Try abbreviation: "BAL"
    - Check ticker pattern: Look for "BAL" in ticker
  </Accordion>
</AccordionGroup>
