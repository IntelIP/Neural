This guide shows how to configure credentials and use TradingClient to place an order.

## Install

Core:

```bash
pip install neural-sdk
```

With trading extras:

```bash
pip install "neural-sdk[trading]"
# Or:
uv add "neural-sdk[trading]"
```

## Credentials

Create a .env and load it:

```bash
# secrets/.env
KALSHI_API_KEY_ID=your_api_key_id
KALSHI_PRIVATE_KEY_BASE64=base64_of_pem   # or
KALSHI_PRIVATE_KEY_PATH=secrets/kalshi_private_key.pem
KALSHI_API_BASE=https://api.elections.kalshi.com
```

```bash
export $(grep -v '^#' secrets/.env | xargs)
```

## Place an Order

```python
from neural.trading import TradingClient

# Initialize using environment creds
client = TradingClient()

# Place an order via the exchange proxy
order = client.exchange.place_order(
    ticker="EXAMPLE",  # replace with a valid ticker
    side="yes",
    count=100,
    price=55
)
```

## WebSocket and FIX (Optional)

For streaming and ultra-low latency:

- WebSocket client: `neural.trading.KalshiWebSocketClient`
- FIX client: `neural.trading.KalshiFIXClient` and `FIXStreamingClient`

These require valid credentials and appropriate permissions. See the streaming docs for best practices.
