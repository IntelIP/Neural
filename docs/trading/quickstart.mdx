## Purpose
Bring up `TradingClient`, authenticate, and submit a Kalshi order through the supported beta path in minutes.

## Prerequisites
- Base install (`pip install neural-sdk`) plus `pip install "neural-sdk[trading]"` (or the equivalent `uv add` command)
- Kalshi API key ID + RSA private key loaded into environment variables as shown below

## Step 1 – Install dependencies

```bash
pip install neural-sdk
pip install "neural-sdk[trading]"
```

If you only need the CLI, docs, or the stable import surface, the base install is enough. Add the `trading` extra when you want Kalshi order entry or streaming helpers; FIX remains experimental.

## Step 2 – Load credentials

```bash
# secrets/.env
KALSHI_API_KEY_ID=your_api_key_id
KALSHI_PRIVATE_KEY_BASE64=base64_of_pem   # or
KALSHI_PRIVATE_KEY_PATH=secrets/kalshi_private_key.pem
KALSHI_ENV=prod
```

```bash
export $(grep -v '^#' secrets/.env | xargs)
```

## Step 3 – Place an order

```python
from neural.trading import TradingClient

client = TradingClient(exchange="kalshi")  # reads env vars
order = client.place_order(
    market_id="EXAMPLE",  # replace with a real ticker
    side="yes",
    quantity=100,
    order_type="limit",
    price=55,
)
print(order)
```

Expect a JSON response containing the Kalshi order ID and status. If you receive a 401, re-check the credential file.

## Optional - Streaming & FIX
- Market data: `neural.trading.KalshiWebSocketClient`, `neural.trading.rest_streaming.RESTStreamingClient`
- Experimental low-latency execution: `neural.trading.KalshiFIXClient` + `FIXStreamingClient`
- FIX remains experimental and requires the `trading` extra.

Consult `trading/market-data` and `trading/fix` for setup details.
